# IIT Delhi Ethics Website

This website provides information about the Institutional Ethics Committee (IEC) and Institutional Animal Ethics Committee (IAEC) at IIT Delhi.

## Pages

- **index.html** - Homepage with navigation and overview
- **institutional-human-ethics.html** - Information about human research ethics
- **institutional-animal-ethics.html** - Information about animal ethics
- **members.html** - List of committee members
- **guidelines.html** - Ethical guidelines and policies
- **forms.html** - Application forms and submission procedures
- **workshop.html** - Workshops and training programs
- **faq.html** - Frequently asked questions

## Getting Started

1. Clone this repository
2. Open `index.html` in your web browser
3. Navigate through the pages using the menu
